Univariate Analysis¶

Bar charts : the simplest visualization chart that maps categories to numbers.

In [27]:
iris_df['Species'].value_counts().plot.bar()
Out[27]:
<matplotlib.axes._subplots.AxesSubplot at 0x1eab81d3860>

Line charts : The categories are represented using a line instead of a bar. Useful when the categories are huge in number like more than 20.

Histograms : A special kind of bar plots that splits the data into even intervals and displays how many rows are in each interval with bars. The only analytical difference is that instead of each bar representing a single value, it represents a range of values.
